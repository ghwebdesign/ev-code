Eview360 & DigitalB Code Guide

Standards for developing flexible, durable, sustainable and consistent code.

Golden rule

Enforce these agreed upon guidelines at all times. Small or large, call out what's incorrect or what you feel should be changed. For additions or contributions to this Code Guide, please contact Darian Rawson.

Every line of code should appear to be written by a single person, no matter the number of contributors.
